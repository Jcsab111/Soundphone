# Soundphone

**Soundphone** is a PyQt5 music & audio player with features like URL streaming, volume control, shuffle, theme customization, and toolbar customization.

## Features

- Play local audio files (`.mp3`, `.wav`, `.ogg`)
- Stream audio from URLs
- Volume control (menu + toolbar)
- Shuffle mode for playback
- Theme selection (default Qt themes or custom `.qss` themes)
- Toolbar customization (add/remove buttons)
- Persistent theme and toolbar settings
- File library with easy removal
- URL playback history

## Usage

1. Open the app.
2. Add audio files via **File > Open File** or stream via **File > Open URL**.
3. Adjust volume from **Volume > Set Volume** or use the toolbar.
4. Enable shuffle via **Settings > Audio > Enable Shuffle**.
5. Customize theme via **Settings > Theme > Select Theme**.
6. Customize the toolbar via **Settings > Toolbar > Customize Toolbar**.
7. View URL history via **History > URL Audio History**.
8. For about info, go to **Help > About**.
9. Visit the repository via **Help > Repo**.
